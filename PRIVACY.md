# Privacy Policy

Term PTT Custom Theme does not collect, sell, or share user data.

The extension stores only local or synced preferences needed to restore the selected appearance:

- Selected colors preset id, name, and color values.
- Selected font id, name, and fallback stack.

The extension reads the active tab URL only to decide whether live preview is available on `https://term.ptt.cc/`.
It does not send browsing history, terminal content, account data, page contents, or appearance preferences to a server controlled by this project.
